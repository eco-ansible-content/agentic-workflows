## NEW: Dual-Mode Operation

### Mode 1: Full Build (New Collection)

**When**: Collection doesn't exist  
**Duration**: 1-3 hours  
**Phases**: 0-9 (all phases)

```bash
Agent({
  prompt: "Build collection from EPIC-2345"
})

# Creates: ~/agentic-workflow-collections/microsoft/scvmm/
# Modules: 0 → 15
# Status: New collection complete
```

### Mode 2: Enhancement (Existing Collection)

**When**: Collection already exists  
**Duration**: 30-60 minutes  
**Phases**: 0, 3 (Enhancement), 7-9

```bash
Agent({
  prompt: "Add modules from EPIC-5678 to microsoft.scvmm"
})

# Enhances: ~/agentic-workflow-collections/microsoft/scvmm/
# Modules: 15 → 18
# Status: 3 modules added, regression tests pass
```

**Benefits**:
- ✅ Preserves existing modules

- ✅ Regression testing
- ✅ Much faster (30-60 min vs 2-3 hours)
