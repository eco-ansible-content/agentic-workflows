# Pattern Recognition Guide

## How to Match Platform to Pattern

### REST API Pattern

**Indicators**:
- "REST API", "HTTP API", "web service"
- "GET/POST/PUT/DELETE endpoints"
- "JSON responses"

**Platforms**: Azure, AWS, SolarWinds, Jira, ServiceNow

### CLI-Based Pattern

**Indicators**:
- "CLI", "command line", "shell"
- "SSH access", "PowerShell cmdlets"
- "show/configure commands"

**Platforms**: Cisco, Linux, Windows, Juniper

### Config File Pattern

**Indicators**:
- "Configuration file", "declarative"
- "YAML/JSON config"
- "Kubernetes manifests"

**Platforms**: Kubernetes, Docker, Terraform
